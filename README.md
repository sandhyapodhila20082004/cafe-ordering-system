Cafe Ordering System

A Flask-based online cafe ordering system that allows customers to:

- View the cafe menu with food images
- Add items to cart
- Place orders with name, phone number, and address
- Track order status
- Generate bills
- Manage orders through an admin dashboard
- View kitchen orders in real time

Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS

Features

- QR Code Menu Access
- Cart Management
- Order Tracking
- Admin Dashboard
- Kitchen Dashboard
- Bill Generation
